Understanding model hallucinations, one span at a time.

What is this?

This project presents a simple interface to view annotated examples from an NLP factuality dataset. Each annotation consists of a source document, a summary, and highlighted nonfactual spans marked by human annotators.

Each span is annotated with:

  • Evidence: Is the claim grounded in the source?
  • Likelihood: Would a model hallucinate this?
  • Consequence: How severe is the hallucination?
  • Common Knowledge: Could this be known without the source?
