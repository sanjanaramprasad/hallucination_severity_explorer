Annotation Explorer

This tool visualizes annotations from our NLP research project on factuality in summarization.

Project Description

Each annotation includes a source document, summary, and spans in the summary marked as nonfactual. Each span is annotated with:

  • Evidence: Is the claim grounded in the source?
  • Likelihood: How likely is it that a model would make this error?
  • Consequence: How severe is this factual error?
  • Common Knowledge: Could this be known without the source?
