About this tool

Hallucination severity methodology

Each annotated hallucinated span includes three key severity attributes measured on a continuous 0.0-1.0 scale:

  • Verifiability (0.0-1.0): Measures how easily the hallucinated claim can be verified against reliable sources. Lower values indicate claims that are difficult or impossible to verify.
  • Plausibility (0.0-1.0): Assesses how believable or realistic the hallucinated content appears. Lower values indicate implausible or obviously false claims.
  • Innocuity (0.0-1.0): Evaluates the potential harm or consequences of believing the hallucinated content. Lower values indicate more dangerous or misleading hallucinations.

Visual indicators

The interface uses intelligently color-coded progress bars to make hallucination severity immediately interpretable:

  • Verifiability bars: Green indicates easily verifiable content, red signals unverifiable or fabricated claims
  • Plausibility bars: Green represents plausible content, red shows implausible or obviously false claims
  • Innocuity bars: Green indicates harmless hallucinations, red highlights potentially dangerous or misleading content

How to use this tool

1

Select annotation type

Choose between human annotations (expert-curated) or LLM annotations (AI-generated) to explore different perspectives on hallucination severity assessment.

2

Compare source and claim

Review the original source document in the left panel alongside the AI-generated summary containing hallucinations in the right panel.

3

Interact with hallucinated spans

Click on highlighted text segments (orange for human annotations, red for LLM annotations) to reveal detailed severity metrics for each hallucination.

4

Analyze severity patterns

Use the precise numerical values and color-coded progress bars to assess hallucination severity across verifiability, plausibility, and innocuity dimensions.
